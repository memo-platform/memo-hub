# Architecture Decision Index

هذه الوثيقة هي مؤشر ADR الرئيسي لمشروع MEMO PLATFORM.

## كيف يُستخدم هذا المؤشر
- يسجل كل ADR مع الحالة والمراجع الرئيسية.
- يتم الحفاظ على نظرة شاملة لجميع قرارات العمارة.
- يجب ربط ADR بالمستندات والدومينات المتأثرة.

## كيفية إضافة ADR جديد
1. إنشاء ملف جديد في `docs/governance/adr/`.
2. استخدم القالب `docs/governance/adr/ADR-0001-template.md`.
3. أضف العنوان والرقم والحالة هنا.
4. اربط ADR بالدومينات والوثائق المتأثرة.

## قواعد السجل
- لا يتم تعديل ADR معتمد بدون تقرير تغيير معماري.
- يتم تسجيل كل قرار معماري مهم بصيغة واضحة ومبررة.
- يجب أن يكون السجل متاحًا لكل أعضاء الفريق وفريق الحوكمة.
