## When to use an RFC before an ADR
An RFC is required when a proposal needs broad review, stakeholder input, or cross-domain alignment before a formal architectural decision is recorded. Use an RFC when the decision is not yet ready for a final ADR, when more collaboration is needed, or when the impacts cross governance and domain boundaries.
