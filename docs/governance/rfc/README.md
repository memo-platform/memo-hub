# RFC Index

This directory contains the Request for Comments (RFC) process for MEMO PLATFORM.

## RFC Listing
- `RFC-0001-template.md` — canonical RFC template.

## RFC Lifecycle
- Draft
- Review
- Approved
- Implemented
- Closed

## RFC → ADR → ACR Relationship
- RFC: used for early proposal, stakeholder review, and discussion before a formal decision.
- ADR: used to record a final architecture decision once the RFC is approved.
- ACR: used to request a change to approved architecture or governance artifacts.

## Status Values
- Draft: Work in progress and not yet under formal review.
- Under Review: The RFC is being evaluated by stakeholders and governance.
- Accepted: The proposal has been approved and is ready for an ADR or implementation.
- Rejected: The proposal was not accepted.
- Superseded: The RFC or decision has been replaced by a newer decision.
